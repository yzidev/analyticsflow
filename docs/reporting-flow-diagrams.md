# Reporting Flow Diagrams

Dokumen ini menggambarkan flow proses reporting di AnalyticsFlow, mulai dari generate manual, initial report dari batch, list/detail/download, sampai flow masing-masing `ReportType`.

Versi diagram visual bisa dibuka langsung di [reporting-flow-diagrams.svg](reporting-flow-diagrams.svg).

![Reporting Flow Diagrams](reporting-flow-diagrams.svg)

## Shared Reporting Components

```mermaid
flowchart TD
    A["ReportController"] --> B["ReportExportService"]
    B --> C{"ReportFormat"}
    C -->|"CSV"| D["CsvReportGenerator"]
    C -->|"XLSX"| E["XlsxReportGenerator placeholder"]
    D --> F["Analytical summary tables"]
    E --> G["Placeholder workbook bytes"]
    F --> H["Write file to data/reports"]
    G --> H
    H --> I["Persist report_metadata"]
    I --> J["Record Micrometer report metrics"]
    J --> K["Return ReportMetadataResponse"]
```

## 1. Manual Report Generation

Endpoint:

```http
POST /api/reports/generate
```

```mermaid
flowchart TD
    A["Client sends GenerateReportRequest"] --> B["ReportController.generate"]
    B --> C["Validate request body"]
    C --> D["ReportExportService.generate"]
    D --> E{"startDate <= endDate?"}
    E -->|"No"| F["Throw BadRequestException"]
    E -->|"Yes"| G["Create reportId and fileName"]
    G --> H["Create report_metadata with status GENERATING"]
    H --> I{"Requested format"}
    I -->|"CSV"| J["CsvReportGenerator.generate(reportType, dates)"]
    I -->|"XLSX"| K["XlsxReportGenerator.generatePlaceholder"]
    J --> L["Write report file under data/reports"]
    K --> L
    L --> M["Update status GENERATED, generatedAt, durationMs"]
    M --> N["Persist report_metadata"]
    N --> O["Record analyticsflow.reports.generated metric"]
    O --> P["Return ReportMetadataResponse with downloadUrl"]
    F --> Q["GlobalExceptionHandler returns ProblemDetail"]
```

## 2. Initial Reports From Batch

Initial reports are generated by the last ETL job step: `GENERATE_INITIAL_REPORT`.

```mermaid
flowchart TD
    A["Spring Batch step GENERATE_INITIAL_REPORT"] --> B["GenerateInitialReportTasklet.execute"]
    B --> C["Read jobId from JobParameters"]
    C --> D["ReportExportService.generateInitialReports(jobId)"]
    D --> E["Loop all ReportType values"]
    E --> F["Generate CSV report for each type"]
    F --> G["Persist report_metadata with jobId"]
    G --> H["Write files to data/reports"]
    H --> I["Set step readCount = report count"]
    I --> J["Set step writeCount = report count"]
    J --> K["Return RepeatStatus.FINISHED"]
```

## 3. List And Detail Reports

Endpoints:

```http
GET /api/reports
GET /api/reports/{reportId}
```

```mermaid
flowchart TD
    A["Client requests reports"] --> B{"Endpoint"}
    B -->|"GET /api/reports"| C["ReportExportService.list"]
    B -->|"GET /api/reports/{reportId}"| D["ReportExportService.get"]
    C --> E["Load report_metadata rows"]
    E --> F["Sort by generatedAt descending"]
    D --> G["Find report_metadata by reportId"]
    G --> H{"Found?"}
    H -->|"No"| I["Throw ResourceNotFoundException"]
    H -->|"Yes"| J["Map entity to ReportMetadataResponse"]
    F --> J
    J --> K["Return metadata response"]
```

## 4. Download Report

Endpoint:

```http
GET /api/reports/{reportId}/download
```

```mermaid
flowchart TD
    A["Client downloads report"] --> B["ReportController.download"]
    B --> C["ReportExportService.get(reportId)"]
    C --> D["Resolve metadata and filename"]
    D --> E["ReportExportService.download(reportId)"]
    E --> F["Resolve file path"]
    F --> G{"Path inside reports directory and file exists?"}
    G -->|"No"| H["Throw ResourceNotFoundException"]
    G -->|"Yes"| I["Return FileSystemResource"]
    I --> J{"ReportFormat"}
    J -->|"CSV"| K["Content-Type text/csv"]
    J -->|"XLSX"| L["Content-Type application/vnd.openxmlformats-officedocument.spreadsheetml.sheet"]
    K --> M["Return attachment response"]
    L --> M
```

## 11. Failure Flow

```mermaid
flowchart TD
    A["Generate report"] --> B["Create metadata status GENERATING"]
    B --> C{"Failure point"}
    C -->|"Invalid date range"| D["BadRequestException before metadata write"]
    C -->|"SQL/runtime failure"| E["Set metadata status FAILED"]
    C -->|"File write failure"| E
    E --> F["Set durationMs and updatedAt"]
    F --> G["Persist failed report_metadata"]
    G --> H["Record failed report metric"]
    H --> I["Throw exception"]
    D --> J["GlobalExceptionHandler returns ProblemDetail"]
    I --> J
```
